# SalesFusion Sales Room Demo Script

**Duration:** 8-10 minutes  
**Audience:** Sales leaders, VPs of Sales, Revenue Operations  
**Goal:** Show how AI qualifies leads automatically and hands off ready buyers

---

## Pre-Demo Setup (30 seconds)

1. **Load Demo Environment**
   - Open Settings panel (gear icon, top right)
   - Click "Load Demo Scenario" button
   - Confirm demo data loaded successfully
   - Close Settings panel

2. **Navigate to Clean State**
   - Start on Sales Room interface (chat view)
   - Clear any existing messages if needed
   - Have browser at 100% zoom for screenshots

---

## Demo Flow

### Scenario 1: Hot Lead - TechFlow Solutions (3 minutes)

**Setup:** Navigate to Transcripts → Select "Sarah Chen - TechFlow Solutions"

**Key Talking Points:**
- "Here's a conversation that just happened with Sarah, VP of Engineering at TechFlow"
- "Watch how our AI captured all four BANT criteria automatically"
- "94% qualification score triggered the handoff popup"

**Walk Through:**
1. **Show Conversation Flow**
   - Point out VP title in first message
   - Highlight budget mention: "budget allocated for this quarter"
   - Show decision authority: "I'm the decision maker for engineering tooling"
   - Note urgency: "choose a vendor by end of this month"

2. **Review AI Summary**
   - Click "View Summary" or scroll to summary section
   - Point out automatically extracted key points
   - Show qualification criteria breakdown (Budget: 95%, Authority: 90%, etc.)
   - Highlight next steps generated by AI

3. **Expected Popup Trigger**
   - Score of 94% should trigger "Talk to Sales" popup
   - Show handoff readiness indicator

**Key Phrases to Emphasize:**
- "Decision maker with budget and timeline"
- "AI caught all qualification signals in real-time"
- "Sales rep gets full context before the call"

## Demo Phrases That Trigger Qualification

Use these exact phrases in live demos to show real-time qualification:

### Budget Triggers:
- "We have budget allocated for this quarter"
- "Cost isn't the main concern if ROI is there"
- "We can invest up to [amount] for the right solution"

### Authority Triggers:
- "I'm the decision maker for [category]"
- "I can approve purchases up to [amount]"
- "Final decision sits with me"

### Need Triggers:
- "Our current process is completely manual"
- "We're losing deals because of slow response times"
- "This is our biggest bottleneck right now"

### Timeline Triggers:
- "We need this implemented by [specific date]"
- "We want to choose a vendor by end of this month"
- "This is urgent - we need to move ASAP"

## Troubleshooting

**If qualification scores seem low:**
- Use exact trigger phrases above
- Mention specific company size (50+, 250+, 1000+ employees)
- Include timeline urgency ("this month", "ASAP")

**If popup doesn't appear:**
- Check qualification score is 75+ in chat store
- Refresh page and try again
- Manually navigate to transcript view to show scoring

**If demo data isn't loading:**
- Open Settings → Click "Load Demo Scenario"
- Check browser console for errors
- Clear sessionStorage and reload
